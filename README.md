# Carbon Footprint Tracker

Веб-сервис для отслеживания личного углеродного следа. Помогает пользователям визуализировать влияние их повседневных привычек (транспорт, энергия, питание) на окружающую среду через расчёт выбросов CO₂, постановку целей и наглядные графики.

## Функциональность

- **Регистрация и авторизация** пользователей
- **Добавление активностей** с автоматическим расчётом выбросов CO₂
- **Погодная корректировка** выбросов для транспортных активностей (интеграция с Open-Meteo API)
- **Дашборд с визуализацией**: круговая диаграмма по категориям, линейный график динамики за 30 дней
- **Установка месячной цели** по выбросам и отслеживание прогресса
- **История записей** с возможностью фильтрации и удаления
- **Административная панель** для управления справочником типов активностей

## Технологический стек

| Компонент | Технология |
|-----------|-----------|
| Backend | Python 3.14, Django 6.0.6 |
| База данных | SQLite (локально), PostgreSQL (продакшн) |
| Frontend | HTML5, CSS3, Bootstrap 5, Chart.js |
| Внешний API | Open-Meteo (погодные данные) |
| Аналитика | Pandas (группировка и агрегация данных) |
| Деплой | PythonAnywhere |

## Источники данных

Коэффициенты выбросов CO₂ взяты из:

1. **Транспорт и топливо:** Приказ Минприроды России от 27.05.2022 №371
2. **Электроэнергия:** Исследование "Carbon Footprint of Electricity Produced in the Russian Federation" (MDPI Energies, 2024)
3. **Продукты питания:** Our World in Data "Environmental impacts of food production"
4. **Общественный транспорт:** EPA Greenhouse Gas Equivalencies Calculator
   

# Как запустить проект локально

1. Клонируйте репозиторий:  
   git clone https://github.com/sunastaaa/carbon-footprint-tracker.git  
   cd carbon-footprint-tracker

2. Создайте и активируйте виртуальное окружение:
   ## Для Windows
   python -m venv venv  
   venv\Scripts\activate

   ## Для macOS/Linux
   python3 -m venv venv  
   source venv/bin/activate

4. Установите зависимости:  
   pip install -r requirements.txt

5. Примените миграции базы данных:  
   python manage.py migrate

6. Создайте суперпользователя:  
   python manage.py createsuperuser

7. Запустите сервер разработки:  
   python manage.py runserver

8. Откройте в браузере: http://127.0.0.1:8000/
